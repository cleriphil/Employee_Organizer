## Employee Organizer

### Description:
Employee Organizer keeps track of employees and their departments and projects. 

## Setup Instructions: 

Run postgres:

  `$ postgres`

In your project directory:

  `$ rake db:create`
  `$ rake db:migrate`
  `$ rake db:test:prepare`
  
Run the app:
  
  `$ ruby app.rb`
